ML Weather Prediction

Developed a weather classification model utilizing data preprocessing, model training, and evaluation, deployed into a web-based application.

Role Machine Learning Developer
Platform Web Application
Tech Stack
Python
Google Colab
Flask

Project Overview

This project focused on applying machine learning algorithms to predict and classify weather conditions based on historical meteorological data. The trained model was integrated into a simple web application using Flask, allowing users to input parameters and receive real-time predictions.

The Problem

Accurate weather prediction at a hyper-local level can be challenging without specialized tools. The goal was to demonstrate how accessible machine learning models could provide reasonable baseline classifications for weather patterns using available datasets, without relying entirely on third-party APIs.

Design Decisions & Architecture

Data was preprocessed and explored extensively within Google Colab to identify key features affecting weather changes. Various algorithms (such as Random Forest and SVM) were evaluated before selecting the best-performing model. The deployment architecture used Flask for its lightweight, straightforward routing capabilities, making it ideal for serving the Python-based model to a web front-end.

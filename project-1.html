IoT Monitoring & Web Dashboard

A comprehensive IoT system featuring real-time sensor data visualization through a responsive web dashboard connected to a MySQL database via REST APIs.

Role Hardware & Web Integrator
Duration 1 Month
Tech Stack
ESP32
MQTT
REST API
MySQL

Project Overview

This project involved designing and deploying a Smart Fan IoT Dashboard that integrates hardware sensors via an ESP32 microcontroller with a responsive web interface. The primary objective was to enable real-time monitoring and remote control of environmental conditions directly from a web browser, bridging the gap between physical hardware and cloud data visualization.

The Problem

Traditional fan and environmental control systems lack intelligent, automated control based on real-time data, leading to energy inefficiency and suboptimal room comfort. Existing solutions were often closed ecosystems that didn't allow for custom dashboard integration or local database management.

Design Decisions & Architecture

The system leverages lightweight communication protocols for low-latency data transfer between the ESP32 hardware and the cloud. The frontend dashboard was built with a clean, modern aesthetic similar to this portfolio, using responsive design principles. Data is processed via a custom REST API and stored in a local MySQL database, which the frontend polls to display real-time metrics using interactive elements.

Key Challenges

One of the main challenges was ensuring reliable synchronization between the hardware state and the web dashboard without overwhelming the server with requests. This was mitigated by optimizing the API endpoints and implementing efficient local database queries.
